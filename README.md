# 山田海斗 自己紹介サイト

クラウドアーキテクト/SRE である山田海斗の、一枚もの（シングルページ）の自己紹介サイト。
GitHub Pages で公開する、**ビルドレスの静的サイト**です。

## デザインコンセプト

**和モダン × テック**。家業の老舗和菓子屋（若松園）というルーツと、クラウドアーキテクト/SRE という現在を結ぶ「和 × テック」。
深い藍の夜・金の差し色・墨流し・和紙・落款（朱の印）・明朝体をキーワードに構成しています。

漢字一文字のテーマを持つ縦スクロールの物語として、以下のセクションで展開します。

| 漢字 | セクション | 内容 |
|---|---|---|
| 序 | Hero | 名前・肩書き（墨滲みで名前が出現、朱の落款スタンプ） |
| 人 | Profile | プロフィール、趣味・特技・好物・苦手 |
| 歩 | Journey | 来歴（1983〜2023の8エピソード、金線が伸びるタイムライン） |
| 技 | Career | 職務経歴（章ごとに技術ロゴが降る） |
| 発 | Output | GitHub/ブログ/Qiita の実績（数字のカウントアップ） |
| 遊 | Zundoko | ズンドコキヨシのターミナル風インタラクティブデモ |

## 技術構成

- **フレームワーク・バンドラ・npm 依存なし。** 静的ファイルのみで動作します。
- スタイル: 純粋な CSS（CSS 変数によるデザイントークン管理）
- スクリプト: バニラ JavaScript（ライブラリなし）
- アニメーション: CSS Scroll-Driven Animations を第一選択とし、非対応ブラウザ向けに IntersectionObserver のフォールバックと `prefers-reduced-motion` 配慮を実装。
- フォント: Google Fonts（Shippori Mincho B1 / Zen Kaku Gothic New / Yuji Syuku）

## ディレクトリ構成

```
.
├── index.html        # ページ本体（全セクションのマークアップ）
├── css/
│   └── style.css     # スタイル一式（デザイントークン・アニメーション）
├── js/
│   └── main.js       # 金粒子生成・フォールバック・カウントアップ・ズンドコデモ
├── images/           # 写真・技術ロゴ（URL安全な半角英数字ファイル名）
│   └── logos/        # 技術ロゴ
├── contents.txt      # コンテンツの元ネタ
├── .nojekyll         # GitHub Pages の Jekyll 処理を無効化
└── CLAUDE.md         # 開発ガイド（デザイン指針）
```

## ローカルでの確認方法

`file://` だと相対パスやフォントで差が出るため、**ローカルサーバー経由**で確認してください。

```bash
# プロジェクトルートで実行
python -m http.server
```

ブラウザで <http://localhost:8000> を開きます。
（ポートを変える場合は `python -m http.server 3000` のように指定）

## デプロイ

GitHub Pages にそのまま push すれば公開されます。ビルド工程は不要です。

## 開発上の注意

詳細なデザイン指針・コーディング規約は [`CLAUDE.md`](./CLAUDE.md) を参照してください。要点は以下の通りです。

- 色は `:root` の CSS 変数を使う（生の hex を書かない）。背景は必ず濃藍系。
- 見出しは明朝体、筆文字は落款と背景の大漢字のみ。
- アニメーションは `transform` と `opacity` のみ。
- 画像ファイル名は半角英数字（日本語ファイル名は使わない）。
- モバイル幅（〜760px）でのレイアウト崩れを確認する。
